Shadersense Introduction:

Shadersense is a Microsoft Visual Studio 2008 plug-in that is designed to provide some common 
intellisense functions for the Shader Language, HLSL. It is fairly simplistic and mimics the intellisense
provided by the default Microsoft Visual Studio IDE.
